Simply Symptoms, a web-based platform that allows patients to input any symptoms they may be experiencing and receive possible conditions as well as suggestions for medications they may take.



To try out the project :
1. create .env file and add the following field
GEMINI_API_KEY=<>
Use this link to get your API key : https://aistudio.google.com/api-keys

3. create a python virtual environment & download from requirements.txt

2. use this command to run the project 
streamlit run streamlit_app.py
